Sistema de gestión de solicitudes de uniformes para el Servicio Nacional de Aprendizaje (SENA) de Colombia. Esta aplicación web permite a los aprendices solicitar uniformes, mientras que administradores, almacenistas y despachadores gestionan el proceso completo de solicitudes, inventario y entregas.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Descripción}{Descripción}}\label{md__r_e_a_d_m_e_autotoc_md1}
Dot\+App SENA es una plataforma web desarrollada con Django que facilita la gestión integral del proceso de solicitud y entrega de uniformes en los centros de formación del SENA. El sistema está diseñado con un modelo de roles que permite diferentes niveles de acceso y funcionalidades según el tipo de usuario.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Características Principales}{Características Principales}}\label{md__r_e_a_d_m_e_autotoc_md2}
